# Qnexy - Visual IVR & WhatsApp Engagement Logic

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Status](https://img.shields.io/badge/status-bootstrap-orange.svg)

**Qnexy** is a specialized communication logic designed to bridge the gap between traditional Chatbots and interactive user experiences. We focus on **Visual IVR** (Interactive Voice Response) and advanced WhatsApp button workflows.

## 🚀 The Core Problem
Most WhatsApp chatbots are limited to text-based menus. Qnexy enables a seamless transition from a message button to a dedicated web interface, allowing for:
- Richer data visualization.
- Instant redirects to custom landing pages (Thank-you pages, payment links, or dashboards).
- Reduced friction in the customer journey.

## 🛠 Key Features (Technical Overview)
- **Visual IVR Integration:** Implementation of systems similar to DialMyApps, redirecting calls/messages to a visual mobile interface.
- **WhatsApp Webhook Handling:** Optimized logic for button-click events and external URL triggering.
- **Dynamic URL Redirects:** Securely passing session data from a chat interface to a web browser.

## 📂 Project Structure
- `/logic`: Core workflow for Visual IVR.
- `/integration`: WhatsApp API connection scripts.
- `/docs`: Technical documentation and implementation guides.

## 📈 Growth & Vision
Currently in its bootstrap phase, Qnexy is being tested in real-world environments to optimize the "Click-to-Web" conversion rate for small and medium businesses.
